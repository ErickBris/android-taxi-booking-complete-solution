Installation Instruction

Guide to setup on Android Studio
  1. Open Android studio and open the cruzer_pro android project.
  2. You have to replace your json url in the following file in android
    • Util/GlobalVariable.java 
      public static final String API = "http://www.jetbench.com/cruzer";
  3. Open AndroidManifest.xml file and replace Google Maps API key under 
    com.google.android.maps.v2.API_KEY
  4. Final Step - Clean and build the project
Guide to setup on Web Admin Panel
  1. Upload the web_sourcefiles files which you got with the purchased to your web hosting server.
  2. Import the database script file (you find it in the database folder) to your MySQL server
  3. Final Step - You have to config the database details in two place
    • web_sourcefiles -> includes -> config.php
    • web_sourcefiles -> core -> db_config.php
